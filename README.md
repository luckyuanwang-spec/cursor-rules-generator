# Cursor Rules Generator

A free online tool to generate `.cursorrules` configuration files for Cursor AI editor. Create custom AI coding rules based on your project description, tech stack, and coding preferences.

**Live Demo**: https://cursor-rules-generator.vercel.app/

## Features

- 🎯 Generate `.cursorrules` files from project description
- 📦 Preset templates for React, Vue, Python, Node.js, and more
- ⚙️ Customizable coding style preferences
- 📋 One-click copy to clipboard
- 🌐 Bilingual support (English/中文)
- 📱 Fully responsive design

## How It Works

1. Select your tech stack
2. Enter project description
3. Choose coding style preferences
4. Click "Generate" to get your `.cursorrules` content
5. Copy and paste into your project root

## Tech Stacks Supported

- React / Next.js
- Vue / Nuxt
- Python / Django / Flask
- Node.js / Express
- TypeScript
- Go
- Rust
- Custom

## Usage

Simply open `index.html` in a browser to use the tool locally.

## Deployment

This tool is deployed on Vercel. Any changes pushed to the main branch will automatically deploy.
